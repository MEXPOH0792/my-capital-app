# My Capital Repo Package

## Что это
Готовый набор для GitHub Pages:
- index.html — рабочее приложение с логикой из исходного файла
- manifest.json
- sw.js
- icon-180.png
- icon-192.png
- icon-512.png
- icon.svg

## Как выложить
1. Создай новый публичный репозиторий на GitHub
2. Загрузи все файлы из этого архива в корень репозитория
3. Открой Settings -> Pages
4. Source: Deploy from a branch
5. Branch: main / root
6. Save
7. Дождись ссылки вида https://USERNAME.github.io/REPO/

## Установка
### iPhone
Открой ссылку в Safari -> Поделиться -> На экран Домой

### Android
Открой ссылку в Chrome -> Установить приложение / Add to Home screen
